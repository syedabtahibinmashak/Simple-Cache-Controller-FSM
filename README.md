# Simple-Cache-Controller-FSM
A simple cache controller simulator using FSM concepts written in C++


### CSE 4305 : Computer Organization and Architecture
### Assignment 02 : Simulating a Simple Cache Controller FSM

### Features:
- Dynamic request queue
- Interactive user input
- State transition simulation with logging
- Cache hit or miss simulation 
- Memory interaction
  
## Installation:
1. Clone the repository:
```bash
git clone https://github.com/syedabtahibinmashak/Simple-Cache-Controller-FSM.git
```
2. Go to the project folder:
```bash
cd Simple-Cache-Controller-FSM
```
3. Compile the code:
```bash
g++ main.cpp -o main
```
4. Run to program:
```bash
./main
```
